## Key Lessons Learned

### DevEco Studio Development
- **Env vars critical**: `DEVECO_SDK_HOME` and `JAVA_HOME` must be set correctly
- `DEVECO_SDK_HOME=C:\Program Files\Huawei\DevEco Studio\sdk`
- `JAVA_HOME=C:\Program Files\Huawei\DevEco Studio\jbr`
- Always rebuild after code changes before installing
- Old HAP was the reason "鸡哥" text didn't show initially

### PowerShell Encoding Issues
- Inline PowerShell commands with Chinese characters fail due to encoding
- Use `.ps1` script files instead of inline `-Command`
- `cmd /c` inside PowerShell gets mangled — use `.bat` files for complex chains
- `2>nul` redirect fails in PowerShell

### Windows UI Automation
- `hdc` has no `input tap` command for HarmonyOS emulator
- Must use Windows mouse events (SetCursorPos + mouse_event) to click emulator
- Multi-position click strategy works — try multiple Y coordinates
- Find emulator window via `Get-Process` where MainWindowTitle = "Emulator"
- Emulator window coords: varies, use `GetWindowRect` to find

### Notification System (WORKING!)
- **Final approach: 3 layers**
  1. System beeps (Console.Beep) for audio attention
  2. Desktop file for visual fallback
  3. Dialog launched via `schtasks` as independent system task (survives parent exit!)
- **Key insight**: Dialog must run as independent process via schtasks, otherwise parent timeout kills it
- Buttons: OK = "我来处理" (stop), Cancel = "你自己搞定" (continue)
- Flag files in `task_flags/` directory
